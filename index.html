Слишком сильное затемнение обложки

  1. Иногда, это портит качество отображения вашей картинки;

Не соблюдение единых отступов у блоков

Не соблюдение единых отступов у блоков

  1. Не нужно делать такие заборчики, ведь любому клиенту явно это будет бросаться в глаза;
  2. С опытом вы сможете делать это ещё аккуратнее.

Как исправить отображение:

  2. Либо скриптом пробросить анимацию в нужные элементы блока (приложу скрипт в доп. материалах).

Отображение картинки автора

Как исправить отображение:

  1. Лучше будет попросить клиента прислать хорошую картинку без таких срезов;

Как один блок показать на Desctop,
а второй для Mobile?

  1. Дублируем блок;
  2. Ставим 1 блоку настройку видимости - "Показать от 760px" (DESCTOP);
  3. Ставим 2 блоку настройку видимости - "Показать до 760px" (MOBILE);
  4. Вносим свои изменения в мобильную версию.

А теперь давайте я это продемонстрирую.

Где править отступы у блоков?

  1. Переходим в "Стиль блока";
  2. Находим параметры "Отступ сверху" и "Отступ снизу";
  3. Заменяем на нужные значения;
  4. Находим полосу, её ещё называют range-slider, distance-slider;
  5. Регулируем кубиками нужные нам отступы слева и справа;
  6. Когда все действия выполнены, нужно сохранить настройку стиля.

А теперь давайте я это продемонстрирую.

Где править отступы у элементов?

  1. Наводим на элемент;
  2. Находим шестерёнку и нажимаем на неё;
  3. Находим параметры "Отступ сверху" и "Отступ снизу";
  4. Заменяем на нужные значения;
  5. Сохраняем.

А теперь давайте я это продемонстрирую.

Где поменять цвет текста / ссылок?

  1. Массовый вариант.
    • Заходим в стили блока;
    • Находим "Цвет фона блока, Цвет текста блока, Цвет ссылок блока";
    • Заменяем значения и сохраняем.
  2. Точечный вариант. Бывает, что не все цвета текста поменялись или же нужно дать цвет определённому элементу.
    • Наводим на элемент и кликаем на шестерёнку;
    • Находим "Цвет текста" и меняем на свой;
    • Сохраняем изменения.

Как изменить размер текста в слайдере?

  1. Для начала находим селектор;
  2. Убеждаемся, что находимся там, где нужно;
  3. Локально меняем размеры;
  4. Переносим код в блок;
  5. Так как мы будем использовать CSS, то дублировать слайдер для адаптивности не нужно, предусмотрим это в коде.

А теперь давайте посмотрим, как это можно реализовать.

Находим селектор

Изменяем размер текста

Код с комментариями:

            /* */ - это обозначение комментариев, браузер не будет их обрабатывать, как код
            @media screen and (max-width: 768px) { /* max-width - макисмальный размер экрана */
              .f-description { /* селектор с описанем отзыва */
                font-size: 17px !important; /* размер шрифта */
              }
            }

В итоге получится такая конструкция:

            @media screen and (max-width: 768px) { 
              .f-description { 
                font-size: 17px !important; 
              }
            }

Добавляем данный код в блок вашего слайдера (раздел "Стиль")

Остался последний пункт, как убрать лишние отступы?

DRY (Не повторяйся)

Если обратить внимание на наш код, то мы видим повторяющиеся блоки.

  1. Старайтесь избегать в коде повторения каких-то блоков кода;
  2. В нашем случае нужно:
    • Оставить только одно @media выражение;
    • Перенести весь код в него.

  2. Добавить CSS в данный блок, чтобы выглядела интереснее;

А теперь давайте посмотрим на код для такой реализации.

HTML код с комментариями:

            button - тег объявления кнопки
            id - идентификатор, по которому будем искать кнопку
            Конструкция:
            <button  id="" >Вверх!</button>

Финальный HTML код:

            <button id="scroll-top-arrow-button">Вверх!</button>

/* CSS код кнопки с комментариями: */

            #scroll-top-arrow-button { /* наш идентификатор с моим названием для кнопки */
              display: inline-block;
              background-color: #FF9800; /* цвет фона */
              width: 50px; /* ширина */
              height: 50px; /* высота */
              text-align: center; /* центруем весь текст внутри, ведь иконка по факту текст */
              border-radius: 4px; /* скругляем */
              margin: 0; /* убираем все отступы */
              position: fixed; /* фиксируем, чтобы она была видна всегда на экране */
              bottom: 15px; /* снизу от экрана отступ 15px */
              right: 90px; /* справа от экрана 90px */
              transition: background-color .3s; /* плавный переход, который нужен при наведении и т.д. */
              z-index: 1000; /* установим, чтобы она была выше по слою */
              border: none; /* убираем рамки */
              font-size: 0; /* установим размер 0, чтобы не было основного текста, иконку добавим дальше */
            }

/* CSS код кнопки при наведении с комментариями: */

            #scroll-top-arrow-button:hover { /* наш идентификатор + :hover(наведение на кнопку) */
              cursor: pointer; /* поменяем на курсор с указательным пальцем */
              background-color: #333; /* поменяем цвет */
            }

/* CSS код кнопки при нажатии с комментариями: */

            #scroll-top-arrow-button:active { /* наш идентификатор + :active(нажатие на кнопку) */
              background-color: #555; /* поменяем цвет */
            }

/* CSS код иконки, которую добавим в кнопку с комментариями: */

            #scroll-top-arrow-button::after { /* наш идентификатор + ::after(псевдоэлемент с иконкой) */
              content: "\f077"; /* добавляем иконку */
              font-family: FontAwesome;  /* установим иконочный шрифт */
              font-weight: normal;  /* сбросим жирность */
              font-style: normal;  /* сбросим начертания */
              font-size: 28px;  /* добавим размер */
              line-height: 50px;  /* добавим высоту строки */
              color: #fff;  /* установим цвет */
            }

/* Финальный CSS код кнопки: */

            #scroll-top-arrow-button {
              display: inline-block;
              background-color: #FF9800;
              width: 50px;
              height: 50px;
              text-align: center;

              margin: 0;

              font-size: 0;

            #scroll-top-arrow-button:hover {
              cursor: pointer;

            }

            #scroll-top-arrow-button:active {

            }

            #scroll-top-arrow-button::after {
              content: "\f077";
              font-family: FontAwesome;
              font-weight: normal;
              font-style: normal;
              font-size: 28px;
              line-height: 50px;
              color: #fff;

/* JS логика, которая будет отвечать за перемещение с комментариями: */

            jQuery(document).ready(function() { /* код начнёт работать, когда страница загрузится */
              var btn = $('#scroll-top-arrow-button'); /* находим нашу кнопку */
              $(window).scroll(function() { /* объявляем, что будет что-то происходить про скроле */
                if ($(window).scrollTop() > 300) { /* если проскролили более 300px */
                  btn.addClass('show'); /* то покажи кнопку */
                } else { /* иначе */
                  btn.removeClass('show'); /* скрой её */
                }
              });
              btn.on('click', function(e) { /* теперь, установим действие при клике на неё */
                e.preventDefault(); /* сбросим все возможные действия */
                $('html, body').animate({scrollTop:0}, '300'); /* если был клик, то с анимацией скроль вверх (300 - скорость) */
              });
            });

/* Финальная JS логика, которая будет отвечать за перемещение: */

            jQuery(document).ready(function() {
              var btn = $('#scroll-top-arrow-button');
              $(window).scroll(function() {
                if ($(window).scrollTop() > 300) {
                  btn.addClass('show');
                } else {
                  btn.removeClass('show');
                }
              });
              btn.on('click', function(e) {
                e.preventDefault();
                $('html, body').animate({scrollTop:0}, '300');
              });
            });

Точечное добавление анимации элементу или блоку (Revealator-Animation)

  1. В 3 уроке мы уже добавляли данную библиотеку;
  2. Теперь решим такие проблему - как добавить анимацию элементам, а не всему блоку?

Рассмотрим проблему

  1. Добавим какой-то список;
  2. И на весь блок добавим класс revealator-slideright;
  3. Далее смотрим на результат.

Как это исправить с помощью скрипта?

  1. Рассмотрим скрипт;
  2. Рассмотрим нужные участки кода;
  3. Удалим ранее добавленный класс;
  4. Придумаем и добавим свой класс, по которому будем искать;
  5. Добавим JS;
  6. Найдём этот список и его элементы;
  7. Вызываем функцию с параметрами.

На самом деле это не так сложно, но давайте я это продемонстрирую :)

Как правильно определить класс объекта при просмотре кода элемента, где перечислены сразу несколько классов?

  1. Кликаем правой кнопкой мышки по элементу
  2. Выбираем просмотр кода элемента
  3. Каждый элемент чаще всего содержит класс и порой их бывает несколько
  4. Один класс может быть основным, а другие модификаторами, которые просто вносят правки с помощью CSS

    • subheader - говорит, что это конкретный подзаголовок
